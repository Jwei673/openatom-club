📌 社团简介

成电开放原子开源社团（开放原子开源社团）是一个致力于推广开源文化、学习开源技术的学生社团。我们相信开源精神可以让技术进步更快、更透明。

🎯 我们的目标

  • 推广开源文化，让更多人了解并参与开源
  • 学习现代软件开发技术，提升成员技术能力
  • 通过实际项目积累经验，培养团队协作能力
  • 建立技术社区，促进知识共享与交流

🛠️ 我们做什么

  • 开发和维护开源项目
  • 举办技术分享会和 Workshop
  • 组织编程竞赛和算法训练
  • 帮助新手入门编程和开源世界

🤝 如何加入我们

我们欢迎所有对开源技术感兴趣的同学加入！无论你是编程新手还是有经验的开发者，都能在我们这里找到适合自己的位置。

📝 加入步骤

  1. 关注我们的 GitHub 仓库 - 点击右上角的 ⭐ Star 表示支持
  2. 加入讨论群 - 联系社团负责人获取 QQ 群/微信群二维码
  3. 完善个人资料 - 在群里修改昵称，介绍自己的技术方向
  4. 选择一个项目 - 从项目列表中选择感兴趣的项目
  5. 开始贡献 - Fork 项目，创建分支，提交你的第一个 PR

💡 我们期待的你

  • 对技术有热情，愿意持续学习
  • 能够使用基本的 Git 命令
  • 有团队协作精神，善于沟通
  • 愿意分享知识，帮助他人成长

🔧 贡献指南

我们非常欢迎各种形式的贡献！无论是代码、文档、bug 报告还是建议，都对我们非常重要。

🐛 报告 Bug

如果你发现了 bug，请通过 GitHub Issues 报告。请提供以下信息：

  • Bug 的清晰描述
  • 复现步骤
  • 预期行为和实际行为
  • 你的环境信息（操作系统、浏览器版本等）

💻 提交代码

  1. Fork 项目到你的 GitHub 账号
  2. 克隆到本地：git clone https://github.com/你的用户名/项目名.git
  3. 创建新分支：git checkout -b feature/你的功能名
  4. 进行开发，确保代码风格一致
  5. 提交代码：git commit -m "Add: 你的功能描述"
  6. 推送到远程：git push origin feature/你的功能名
  7. 创建 Pull Request，等待 Code Review

📚 Git 使用教程

如果你不熟悉 Git，不用担心！下面是入门指南。

🔰 Git 基础命令

# 克隆仓库
git clone https://github.com/Jwei673/openatom-club.git

# 进入项目目录
cd openatom-club

# 查看当前状态
git status

# 查看修改内容
git diff

# 添加文件到暂存区
git add 文件名

# 提交更改
git commit -m "提交信息"

# 推送到远程仓库
git push origin 分支名

# 拉取最新代码
git pull origin 分支名

🌿 Git 分支管理

# 查看所有分支
git branch

# 创建新分支
git checkout -b feature/新功能

# 切换分支
git checkout 分支名

# 合并分支
git merge 分支名

# 删除分支
git branch -d 分支名

❓ 常见问题

Q: 我是编程新手，能加入吗？

A: 当然可以！我们欢迎所有对技术感兴趣的同学。社团内有经验丰富的成员可以带你入门。

Q: 加入社团需要交费吗？

A: 不需要。我们是一个纯技术兴趣社团，不收取任何费用。

Q: 我没有 GitHub 账号怎么办？

A: GitHub 账号是免费注册的。访问 github.com 点击 Sign up 开始注册。

Q: 我可以同时参与多个项目吗？

A: 可以！根据你的时间和精力，选择参与一个或多个项目。

Q: 贡献代码有什么好处？

A: 你可以积累项目经验、学习协作开发、提升技术能力、结识志同道合的朋友，并且你的贡献会被永久记录在项目中！
