## 🌟 简介

成电开放原子开源社团（开放原子开源社团）是一个致力于推广开源文化、学习开源技术的学生社团。我们相信开源精神可以让技术进步更快、更透明。

## 🎯 我们做什么

- 💻 开发和维护开源项目
- 📚 举办技术分享会和 Workshop
- 🤝 组织编程竞赛和算法训练
- 🌱 帮助新手入门编程和开源世界

## 📂 项目结构

```
openatom-club/
├── index.html      # 首页
├── members.html    # 成员页面
├── projects.html   # 项目页面
├── style.css       # GitHub 深色主题样式
├── docs/
│   └── index.html  # 文档中心
└── README.md       # 项目说明
```

## 🛠️ 技术栈

- **前端**: HTML5, CSS3（GitHub 深色主题）
- **部署**: GitHub Pages（免费托管）
- **协作**: Git, GitHub

## 🚀 快速开始

### 查看网站

网站已部署到 GitHub Pages，直接访问：

**[https://Jwei673.github.io/openatom-club](https://Jwei673.github.io/openatom-club)**

### 本地预览

1. 克隆仓库
```bash
git clone https://github.com/Jwei673/openatom-club.git
```

2. 进入目录
```bash
cd openatom-club
```

3. 直接用浏览器打开 `index.html` 文件即可预览

## 🤝 如何参与

我们欢迎所有对开源技术感兴趣的同学加入！

### 加入步骤

1. ⭐ Star 本项目表示支持
2. 📋 查看 [Issues](https://github.com/Jwei673/openatom-club/issues) 选择感兴趣的任务
3. 🍴 Fork 本仓库到你的账号
4. 💻 克隆到本地，创建新分支进行开发
5. 📬 提交 Pull Request，等待 Code Review

### 贡献指南

- 提交代码前请确保本地测试通过
- 提交信息请使用清晰的描述（英文或中文均可）
- 如果有新的功能想法，欢迎先提 Issue 讨论
